Цель 

	•	Максимум заявок на обратный звонок.
	•	Никаких карт, адресов и лишних контактов — только страница контактов с нашими контактами и форма «Давайте мы вам позвоним» (в модальном окне).

Общий стиль
	•	Темная, глубокая сцена: фон — чёрный/темно-синий градиент без лишних паттернов.
	•	Визуальный язык — VR/неон, «электрический» синий + голубой.
	•	Акценты: мягкое свечение/глоу, пульсация синего на ключевых элементах (кнопки/контуры).
	•	Атмосфера — «технологично, чисто, футуристично» (никакой кислотной пестроты).

Палитра (референс)
	•	Фон: #00040E → #0A1022 (радиальный/диагональный градиент).
	•	Акцент 1 (электрический синий): #00E5FF.
	•	Акцент 2 (глубокий синий): #2B6CFF.
	•	Подсветка/глоу: полупрозрачные слои этих же оттенков.
	•	Текст основной: #FFFFFF, вторичный: #AAB3C5.

Типографика
	•	Современный гротеск: Inter/Manrope/SF Pro (что доступнее).
	•	Заголовки: полужирные; текст — нормальный; межстрочные увеличенные для «воздуха».

Компоновка вкладки
	•	Центрированный блок: заголовок + кнопка-CTA.
	•	Заголовок: «Давайте мы вам позвоним».
	•	Подзаголовок: «Оставьте свой номер, и мы перезвоним вам».
	•	Под ними — крупная кнопка «Давайте мы вам позвоним».

Поведение (UX)
	•	Клик по CTA открывает модальное окно с формой.
	•	Закрытие модалки: крестик, клик по оверлею, клавиша Esc; фокус — внутри модалки (trap).
	•	Легкая неоновая пульсация CTA (cycle 2–3s), hover — усиление свечения, press — лёгкая компрессия.

Форма в модалке

Тексты полей:
	1.	Ваше имя — опционально, placeholder: «Введите имя».
	2.	Ваш номер телефона* — обязательное поле, placeholder: «+7 (___) ___-__-__».
	3.	Перезвонить в — селектор времени (выпадающий список).

Под формой:
— строка мелким текстом: «Вы даете согласие на обработку персональных данных.» (желательно кликабельная ссылка на политику).

Кнопка отправки:
— текст «Жду звонка», широкая, со свечением и hover-эффектом.

Селектор времени
	•	Значения каждые 10–15 минут, рабочий диапазон 10:00–22:00 по МСК (можно уточнить позже).
	•	По умолчанию предустановить 14:40 (как в твоём примере).
	•	Если текущее время позже выбранного — подсказка: «Позвоним завтра в выбранное время / ближайшее рабочее».

Валидация и маска телефона
	•	Маска ввода (желательно): +7 (___) ___-__-__.
	•	Разрешать международные форматы +375 … и т. п. (паттерн типа: ^\\+?[\\d\\s().-]{10,}$).
	•	Ошибки показывать под полем, лаконично:
	•	«Введите номер телефона» (пусто)
	•	«Проверьте формат номера» (некорректно)

Состояния
	•	Ошибки: красная тонкая подсветка поля + текст ошибки; без «красной сирены».
	•	Отправка: кнопка в состоянии «loading» (спиннер, текст «Отправляем…»).
	•	Успех: заменяем содержимое модалки на screen «Спасибо! Мы перезвоним в {время}» + кнопка «Ок».
	•	Сбой: «Не получилось отправить. Попробуйте ещё раз или позже.»

Анимации (ненавязчивые)
	•	CTA: пульсирующее свечение (keyframes: лёгкое увеличение тени/яркости).
	•	Инпуты/селект: при фокусе — неоновая кайма и едва заметный inner-glow.
	•	Модалка: scale-in + fade-in (300–400 ms).
	•	Hover кнопок: усиление glow, лёгкий подъём (-2px).

Адаптив
	•	Мобайл first: модалка во всю ширину/высоту, крупные поля, тач-зоны ≥ 44px.
	•	Десктоп: контейнер формы ~360–420 px шириной, радиусы 20–24 px, большие поля.
	•	Без горизонтального скролла на всех брейкпоинтах.

Доступность
	•	Семантические <label> связаны с input id.
	•	aria-modal, role="dialog", фокус на первом поле, Esc закрывает.
	•	Контраст текста/фона ≥ WCAG AA.
	•	Подписи ошибок привязаны к полям (aria-describedby).

Технические требования
	•	Чистые HTML + CSS + Vanilla JS, без фреймворков и лишних библиотек.
	•	Лёгкие тени/градиенты без тяжёлых фильтров, чтобы не грузить GPU.
	•	Минимум внешних шрифтов; font-display: swap.
	•	Код разбит: стили в <style> или отдельный .css, JS — внизу страницы/отдельный .js.

Интеграция формы (когда подключишь бэкенд)
	•	Отправка POST /api/callback (JSON):

{ "name": "Имя", "phone": "+7 ...", "time": "14:40", "page": "/contacts" }


	•	Добавить: скрытое honeypot-поле для антиспама, UTM-метки, referrer.
	•	События аналитики: callback_submit, callback_success, callback_error.

Тексты (готовые)
	•	Заголовок страницы: «Давайте мы вам позвоним»
	•	Подзаголовок: «Оставьте свой номер, и мы перезвоним вам»
	•	Поле 1: «Ваше имя»
	•	Поле 2: «Ваш номер телефона»*
	•	Поле 3: «Перезвонить в»
	•	Кнопка: «Жду звонка»
	•	Согласие: «Вы даете согласие на обработку персональных данных.»
	•	Успех: «Спасибо! Мы свяжемся с вами в {время}.»

Нюансы/детали визуала
	•	Карточка модалки — glass/neo-glass: тёмный полупрозрачный фон rgba(15,22,41,0.9) + неоновая обводка (внешнее свечение).
	•	Радиусы: 20–24px, тени — мягкие, шириной 20–40px, прозрачность невысокая.
	•	Линии-разделители (если понадобятся) — тонкие «голографические» (1px с лёгким свечением).
	•	Никаких лишних иконок/соцсетей — чтобы не уводить от цели.

Критерии приёмки
	•	Вкладка без карты/адресов, только CTA и модальная форма.
	•	Цвета/свечение соответствуют VR-стилю; есть пульсация синего 
	•	Селектор времени присутствует
	•	Валидация телефона работает, ошибки выводятся корректно.
	•	Доступность: управление с клавиатуры, закрытие по Esc.
	•	Адаптив: мобильная и десктопная версии выглядят одинаково стильно.
	•	Успешная отправка показывает «Спасибо»-экран.
